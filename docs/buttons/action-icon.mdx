# ActionIcon

Primary action icon button to trigger an operation. It's a simple button that contains no text but only icon.

```tsx
import { ActionIcon } from "rizzui";
```

<div className="mb-20 md:mb-28" />

The default style of **ActionIcon** component.

You can set the loading state of the **ActionIcon** button using **isLoading** property.

### ActionIcon Props

Here is the API documentation of the ActionIcon component. And the rest of the props are the same as the original html button. You can use props like **id**, **title**, **onClick**, **onFocus**, **onBlur** etc.

<div className="overflow-x-auto">

|   Props   |                        Type                        |                                         Description                                         |   Default   |
| :-------: | :------------------------------------------------: | :-----------------------------------------------------------------------------------------: | :---------: |
|    as     |                 `button` or `span`                 |                                          Render as                                          | `"button"`  |
| children  |                 `React.ReactNode`                  |                                      Use any SVG icon                                       |    \_\_     |
|   type    | [ActionIconButtonTypes](#action-icon-button-types) |                            Set the original HTML type of button                             | `"button"`  |
|  variant  |     [ActionIconVariants](#actionicon-variants)     |                                 Set the ActionIcon variants                                 |  `"solid"`  |
|   size    |        [ActionIconSizes](#actionicon-sizes)        | Set the size of the component. <br /> `"sm"` is equivalent to the dense ActionIcon styling. |   `"md"`    |
|  rounded  |      [ActionIconRounded](#actionicon-rounded)      |                        Set border radius of the ActionIcon component                        |   `"md"`    |
|   color   |       [ActionIconColors](#actionicon-colors)       |                                   Change ActionIcon color                                   | `"primary"` |
| isLoading |                     `boolean`                      |                            Set the loading status of ActionIcon                             |    \_\_     |
| disabled  |                     `boolean`                      |                              Disabled state of the ActionIcon                               |    \_\_     |
| className |                      `string`                      |                             Add custom classes for extra style                              |    \_\_     |
|    ref    |              `Ref<HTMLButtonElement>`              |                                         forwardRef                                          |    \_\_     |
|    ...    |     `ButtonHTMLAttributes` or `HTMLAttributes`     |                native props like **onClick**, **title**, **aria-label** ...                 |    \_\_     |

</div>
